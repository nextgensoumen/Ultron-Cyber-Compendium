# Security Policy

## Supported Versions

The Ultron-Cyber-Compendium purely serves as a documentation repository. We do not provide actively running software releases, executables, or services. However, any scripts, automation snippets, or documentation found tracking the `main` branch is actively maintained.

| Repository Content | Supported          |
| ------------------ | ------------------ |
| `main` branch      | :white_check_mark: |
| Historical Commits | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within the Ultron Cyber Compendium documentation, website, automated scripts, or infrastructure workflows, please do not disclose it publicly on issues or pull requests.

Instead, please send an email to the repository maintainers. Be sure to include:
- A description of the vulnerability or dangerous configuration
- The exact file and line number
- Potential impact or exploitation scenario
- Suggested remediation

We take all security reports seriously and promise to investigate and respond promptly. When the issue is resolved, we will publish a coordinated fix.
