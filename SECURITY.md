## 🏷️ Supported Versions

The Ultron-Cyber-Compendium purely serves as a documentation repository. We do not provide actively running software releases, executables, or services. However, any scripts, automation snippets, or documentation found tracking the `main` branch are actively maintained.

| 🗂️ Repository Content | 🛡️ Support Status | ℹ️ Description |
|:---|:---:|:---|
| **`main` branch** | ✅ Supported | Primary repository tracking up-to-date documentation and scripts. |
| **Historical Commits** | ❌ Unsupported | Past commits and deprecated documentation instructions. |

---

## 🚨 Reporting a Vulnerability

**DO NOT** disclose security vulnerabilities publicly on GitHub issues or pull requests. 

If you discover a security vulnerability within the Ultron Cyber Compendium documentation (such as a dangerously architected script, insecure recommended configuration, or flaw in our internal GitHub action workflows), please report it responsibly.

### 📝 How to Report

Please send an email directly to the repository maintainers. Be sure your report includes the following details:

1. **Description:** A detailed description of the vulnerability or dangerous configuration.
2. **Location:** The exact file name and line number(s) involved.
3. **Impact:** The potential impact or exploitation scenario if a user follows the documentation.
4. **Remediation:** Suggested fixes or secure configurations.

> **Our Promise:** We take all security reports seriously and promise to investigate and respond promptly. When the issue is resolved, we will publish a coordinated fix and credit you for the responsible disclosure.
